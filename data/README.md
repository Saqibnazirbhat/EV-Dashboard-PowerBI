# Dataset — Electric Vehicle Population

This folder contains the source data used by the dashboard, provided compressed as
`Electric_Vehicle_Population_Data.zip` (≈35 MB uncompressed CSV, 150,000+ rows).

**Source:** [Electric Vehicle Population — Kaggle](https://www.kaggle.com/datasets/willianoliveiragibin/electric-vehicle-population)
(originally published by the Washington State Department of Licensing).

To use it, unzip the file to obtain `Electric_Vehicle_Population_Data.csv`.

## Data Dictionary

| Column | Description |
| --- | --- |
| `VIN (1-10)` | First 10 characters of the Vehicle Identification Number |
| `County` | County where the vehicle is registered |
| `City` | City where the vehicle is registered |
| `State` | State (predominantly WA) |
| `Postal Code` | Registration postal code |
| `Model Year` | Model year of the vehicle |
| `Make` | Manufacturer (e.g., Tesla, Nissan) |
| `Model` | Vehicle model |
| `Electric Vehicle Type` | Battery Electric Vehicle (BEV) or Plug-in Hybrid Electric Vehicle (PHEV) |
| `Clean Alternative Fuel Vehicle (CAFV) Eligibility` | CAFV eligibility status |
| `Electric Range` | Electric-only driving range, in miles |
| `Base MSRP` | Base manufacturer's suggested retail price |
| `Legislative District` | Legislative district of registration |
| `DOL Vehicle ID` | Department of Licensing unique vehicle identifier |
| `Vehicle Location` | Geographic point (longitude/latitude) |
| `Electric Utility` | Electric utility provider(s) serving the location |
| `2020 Census Tract` | 2020 census tract identifier |
